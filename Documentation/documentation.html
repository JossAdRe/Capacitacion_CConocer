Included Files

  • Engage v1.x - Template files
    • css - CSS files
      • Vendor - All css plugin that are used in the template
      • scss - All preprocessor CSS
        • Global - Global preprocessor CSS folder
          • _base.scss - css which includes all the basics used by this template
          • _variable.scss - variable color used by this theme
      • section - Section preprocessor CSS
        • _section-about.scss - css section about
        • _section-contact.scss - css section contact
        • _section-counter.scss - css section counter
        • _section-faq.scss - css section faq
        • _section-footer.scss - css section footer
        • _section-gap.scss - css section gap
        • _section-header.scss - css section header
        • _section-hero.scss - css section hero
        • _section-portfolio.scss - css section portfolio
        • _section-pricing.scss - css section pricing
        • _section-service.scss - css section service
        • _section-testimonial.scss - css section testimonial
        • _section-wyhus.scss - css section testimonial
      • main.css - main theme style
    • javascript - JS files
      • Vendor - All javascript plugin that are used in the template
      • main.js - main js file
    • asset - All the neccessary source files of the template
      • client - client image logo
      • hero - hero banner section image
      • page - image and content that used such as logo brand
      • random - image used in page
      • user - user image file
    • 1. home pages - Home Pages
  • Documentation - Documentation files
